# Custom Theme

Demonstrates fully customizing the cursor overlay and keystroke HUD appearance using a code editor page.

## Features demonstrated

- Custom `cursor` SVG file (`cursor.svg`)
- Increased `cursorSize` (32px)
- HUD `background` with custom color and opacity
- HUD `color`, `fontSize`, and `fontFamily` overrides
- HUD `borderRadius` customization
- HUD `position` set to `"top"` (default is `"bottom"`)

## Run

```bash
cd examples/custom-theme
webreel record
```
