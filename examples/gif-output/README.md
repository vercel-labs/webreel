# GIF Output

Demonstrates outputting the recording as an animated GIF instead of the default MP4.

## Features demonstrated

- `output` field set to a `.gif` extension
- GIF encoding (palette-based, 15fps)

## Run

```bash
cd examples/gif-output
webreel record
```

The output `videos/gif-output.gif` can be embedded in GitHub READMEs, docs, or chat messages.
