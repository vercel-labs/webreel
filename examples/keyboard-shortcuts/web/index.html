Release Notes - v2.0

Draft - Last edited 2 minutes ago

We are excited to announce the release of version 2.0, our biggest update yet. This release includes a redesigned dashboard, improved performance across the board, and several features our community has been requesting.

Redesigned Dashboard - The new dashboard loads 3x faster and includes customizable widgets, real-time metrics, and a unified search bar that lets you find anything across your projects.

Collaboration - Team members can now edit documents simultaneously with live cursors, inline comments, and version history. Changes sync instantly across all connected clients.

We have also improved our API response times by 40% and added support for webhooks, bulk operations, and granular rate limiting. See the migration guide for details on upgrading from v1.x.
