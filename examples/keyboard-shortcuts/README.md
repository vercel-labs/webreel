# Keyboard Shortcuts

Demonstrates pressing key combos and displaying them in the keystroke HUD overlay. Uses a code editor page as the target.

## Features demonstrated

- `key` action with modifier combos (`cmd+a`, `cmd+c`, `cmd+v`, `cmd+z`)
- `key` action with single keys (`Escape`)
- Custom `label` for human-readable HUD text
- Keystroke HUD overlay display

## Run

```bash
cd examples/keyboard-shortcuts
webreel record
```
