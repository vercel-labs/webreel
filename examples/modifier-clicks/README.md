# Modifier Clicks

Demonstrates clicking elements with modifier keys held down, simulating multi-select in a file manager.

## Features demonstrated

- `click` action with `modifiers: ["cmd"]` for additive selection
- `click` action with `modifiers: ["shift"]` for range selection
- `click` action with `within` to scope element search to a container
- Interactive file selection with visual feedback

## Run

```bash
cd examples/modifier-clicks
webreel record
```
