# Screenshots

Demonstrates capturing PNG screenshots at specific points during a recording. Useful for generating static marketing assets or documentation images alongside videos.

## Features demonstrated

- `screenshot` action with `output` path
- Taking multiple screenshots at different stages of a flow
- Combining screenshots with other actions (`click`, `scroll`)

## Run

```bash
cd examples/screenshots
webreel record
```

Screenshots are saved as PNG files in the working directory.
